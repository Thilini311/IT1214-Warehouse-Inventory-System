# Warehouse Inventory Management System

## 📌 Description
This is a simple Java-based Warehouse Inventory Management System using Java Collections Framework (HashMap). It allows users to manage inventory efficiently through a console-based interface.

## ⚙️ Features
- Add new items
- Remove items
- Update item quantity
- Search items by ID or name
- Display all inventory items

## ▶️ How to Run

1. Compile all Java files:
   javac *.java

2. Run the program:
   java Warehouse
